# How to View and Track Flags

## Quick Answer
You can check if flags are created in **4 different ways**:

---

## 1. **View All Flags (Dedicated View)**
```bash
./.venv/bin/python run_app.py --flags
```

**Output:**
```
======================================================================
FLAGS REGISTRY
======================================================================

✓ Total flags: 1

GENERATED FLAGS (1)
----------------------------------------------------------------------

[1] Create_AI_pipeline_flag
    Expression: IF(SUM(Sales[Sales]) > 1000, "Yes", "No")
    Description: Create AI pipeline flag
    Created: 2026-03-18T21:38:36.152345

======================================================================
```

✅ **Best for:** Specifically checking if any flags were created

---

## 2. **View Full Registry (All Items)**
```bash
./.venv/bin/python run_app.py --registry
```

**Output:**
```
======================================================================
FULL REGISTRY (7 items)
======================================================================

MEASURES (4)
----------------------------------------------------------------------
  ✓ Create_total_sales_measure (generated)
  ○ Total_Sales (existing)
  ○ Total_Quantity (existing)
  ○ Average_Price (existing)

FLAGS (1)
----------------------------------------------------------------------
  ✓ Create_AI_pipeline_flag (generated)

TABLES (1)
----------------------------------------------------------------------
  ✓ Create_top_5_products_table_by_total_sales (generated)

======================================================================
```

✅ **Best for:** See all items across all types (flags, measures, tables, columns)

---

## 3. **Filter Registry by Type**
```bash
# View only flags
./.venv/bin/python run_app.py --list-by-type flag

# View only measures
./.venv/bin/python run_app.py --list-by-type measure

# View only tables
./.venv/bin/python run_app.py --list-by-type table

# View only columns
./.venv/bin/python run_app.py --list-by-type column
```

✅ **Best for:** Filter items by specific type

---

## 5. **Interactive Mode (Create Custom Flags)**
```bash
./.venv/bin/python run_app.py --interactive
```

Then:
1. Select `flag` as the item type
2. Describe your flag (e.g., "Create high sales flag where Sales > 5000")
3. Press Enter
4. View your created flag

✅ **Best for:** Creating custom flags interactively

---

## What the Output Symbols Mean

| Symbol | Meaning |
|--------|---------|
| `✓` | Flag was generated by the AI |
| `○` | Flag already existed in model |
| `✗` | No flags found |

---

## Quick Status Checks

### Check if ANY flags exist:
```bash
./.venv/bin/python run_app.py --flags
```
If you see `✗ No flags created yet.` → No flags exist

If you see `✓ Total flags: X` → X flags have been created

---

### Check which flags are generated vs existing:
```bash
./.venv/bin/python run_app.py --list-by-type flag
```

Look at the status after the name:
- `(generated)` = Created by the AI assistant
- `(existing)` = Already existed in your model

## Examples

### Example 1: Create a flag, then view it
```bash
# Create a flag in interactive mode
./.venv/bin/python run_app.py --interactive

# View all flags
./.venv/bin/python run_app.py --flags

# View just flags
./.venv/bin/python run_app.py --list-by-type flag
```

### Example 2: See everything created in the session
```bash
./.venv/bin/python run_app.py --registry
```

### Example 3: Count how many flags were generated in demo
```bash
./.venv/bin/python run_app.py --demo

# Then view only generated flags
./.venv/bin/python run_app.py --flags
```

## Programmatic Access (In Python)

If you're using the agent in code:

```python
from assistant_app.cli import build_agent

agent = build_agent()

# Get all flags
flags = agent.registry.get_items_by_type('flag')
print(f"Total flags: {len(flags)}")

# Print flags summary
print(agent.registry.flags_summary())

# Get all items
all_items = list(agent.registry.items.values())
print(f"Total items: {len(all_items)}")
```
