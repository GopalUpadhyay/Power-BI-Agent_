  - Expected Confidence: High (85-95%)
  - Test: Create combined table of SalesPerson + Targets

================================================================================
WHAT TO CHECK IF TESTS FAIL
================================================================================

Issue: Relationships not detected
→ Check: Are column names exactly matching? (case-sensitive)
→ Check: Do join key columns have same data type?
→ Check: Are there any NULL values in join keys?
→ Check: Open browser console (F12) for error logs

Issue: Wrong row count in combined table
→ Check: Are you using INNER join (only matches) or LEFT join (all from left)?
→ Check: Are there duplicate values in join keys?
→ Check: Count the expected matches manually

Issue: Edit/Delete not working
→ Check: Any console errors?
→ Check: Refresh browser and try again
→ Check: Verify files haven't been deleted

Issue: Data type mismatch
→ Check: One table has numeric IDs, other has text?
→ Try: Manually fix by converting one to match the other
→ Or: Update your CSV files before uploading
