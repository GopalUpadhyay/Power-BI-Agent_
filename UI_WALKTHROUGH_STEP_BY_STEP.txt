OPTIONAL: DELETE OR EDIT MODELS, FILES, & RELATIONSHIPS
═════════════════════════════════════════════════════════════════════════

DELETE A MODEL:
  Location: "Available Model Cards" section
  Look for: 🗑️ button next to model
  Action: 
    1. Click 🗑️ button once (warning appears: "Click again to confirm")
    2. Click 🗑️ button second time to confirm
  Result: Model is permanently deleted with all its files

DELETE AN UPLOADED FILE:
  Location: "Uploaded files" section (after "Active Model" buttons)
  Look for: 🗑️ button in each row
  Action:
    1. Click 🗑️ button next to file to delete
    2. Confirm the action
  Result: File is removed, but metadata is preserved

EDIT RELATIONSHIPS:
  Location: "Relationship Validation & Editing" table
  How to Edit:
    1. Click on any cell in the table
    2. Type new value
    3. Click [ 💾 Save Relationship Edits ] when done
  
  How to Delete a Relationship:
    1. Leave entire row blank (clear all 5 cells)
    2. Click [ 💾 Save Relationship Edits ]
    3. That relationship row will be deleted

DELETE ALL RELATIONSHIPS:
  Location: Below the relationship table
  Button: [ 🗑️ Clear All Relationships ]
  Action:
    1. Click once (warning appears: "Click again to confirm")
    2. Click again to confirm
  Result: All relationships deleted (can re-detect anytime with 🔗 button)

═════════════════════════════════════════════════════════════════════════
TROUBLESHOOTING - IF SOMETHING GOES WRONG:
═════════════════════════════════════════════════════════════════════════

Problem: "No files selected when I click Store Uploaded Files"
Solution:
  → Make sure files are actually selected (names visible)
  → Try selecting files one at a time first


Problem: "Identified 0 relationships"
Solution:
  → Check if column names match exactly (EmployeeID in both tables)
  → Click "🔁 Re-Detect Relationships" to scan again
  → Or manually add the relationship using the data editor


Problem: "Error about relationships not found"
Solution:
  → Click "🔁 Re-Detect Relationships"
  → If still nothing, manually edit the relationship grid
  → Add row with: from_table=Sales, from_column=EmployeeID,
                  to_table=SalesPerson, to_column=EmployeeID


Problem: "File upload not working"
Solution:
  → Reload page: Ctrl+R (or Command+R)
  → Check file format (must be CSV, JSON, PBIX, or image)
  → File size too large? Try smaller files first


Problem: "Model not appearing in dropdown"
Solution:
  → Reload the page
  → Click "🔄 Reload Active Model" button
  → Or close and reopen the browser
